Introduccion a lo que llevamos aprendido de CSS

Para usar CSS es necesario que primero creemos un archivo en nuestro documento con la extension ".css",
luego de eso es neceario incluirlo en nuestro archivo HTML o en los que vayamos a requerir los estilos,
de igual forma puedes crear varios para diferentes archivos que visualices vayan a ser diferentes pero
si piensas mantener el mismo estilo en toda tu pagina web, te recomiendo hacer contendores con <div>.

Para incluir nuestro archivo css en nuestro archivo HTML te recomiendo usar el siguiente codigo:

< link rel="stylesheet" href="nombre.css"/>


Para poner de color azul el texto de arriba encerre la expresion en un contenedor <div> de esta forma:
<div style="color: rgb(100, 100, 131);">< link rel="stylesheet" href="nombre.css"/>"</div>



A continuacion veremos la sintaxis para usar CSS

                .selector{
                    propiedad: valor;
                    propiedad: valor;
                    propiedad: valor
                }

Se basa en una regla por linea terminada por punto y coma, no es necesario ponerle punto y coma la ultima
regla pero es opcional, no pasa nada si se la pones o no.
