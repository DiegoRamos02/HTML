Introduccion de lo que hemos aprendido

Con lo que llevamos del curso hemos aprendido que todo documento HTML inicia con un "!DOCTYPE html" seguido de un "html",
tengamos en cuenta que estas sentencias que he mencionado llevan un "<" antes y un ">" despues, al igual que "html" y otras etiquetas
llevan un cierre que seria un "<" seguido de "/" , "-sentencia-" y ">". Tambien sabemos que head es donde definimos los metadatos
como el "charset" de tipo "UTF-8" y el titulo de la pagina web para poder identifacarla de manera rapida.

Para el caso de hacer enlaces que te permitan moverte de pagina en pagina o en la misma pero diferente apartado usaremos lo siguiente...

<a href = "ejemplo.html o una url"> "Descripcion" /a>

Ahora para hacer contenedores seria tipo:
<div class="Nombre" style="una funcion"/> "Lo que vayas a englobar" </div>
Un ejemplo de contenedor fue lo que hicimos para los enlaces que se muestran en la parte de arriba.

Viendo lo aprendido a la vez que mostramos como usar listas

    Elemtos vistos hasta el momento para saber HTML

  • El < !DOCTYPE html > (aqui se define el tipo de documento).

  • El < html > < /html > (se define la etiqueta html).

  • El < head > < /head > (donde se definen los metadatos y nombre).

  • El < body > < /body > (aqui se encuenytran los elementos visibles).

  • El < h1 > < /h1 > (sabemos que hay hasta h6 y son encabezados).

  • El < p > < /p > (es para escribir parrafos).

  • El < pre > < /pre > (es para escribir parrafos con salto de linea automatico).

  • El < hr > (sirve para separar por secciones la pagina).

  • El < ul > < /ul > (para hacer listas como esta, ul es desordenada).

  • El < ol > < /ol > (para hacer listas ordenadas).

  • El < div > < /div > (se utiliza para hacer contendores).

  • El < b > < /b > (sirve para mostrar texto en negritas).

  • El < form > < /form > (sirve para hacer formularios).

  • El < label > < /label > (sirve para ponerle nombre al campo).

  • El < input > < /input > (sirve para poner el campo, pero aqui se pondria: type="text" name="nombre").

  • El < select > < /select > (sirve para desplegar listas de opciones).

      • Aqui veremos una lista ordenada y los diferentes caracteres que se pueden usar
    1. Si quieres que sean numeros solo basta con <li>...</li>
    2. Lista con letras mayusculas <li TYPE="A">...</li>
    3. Lista con letras minisculas TYPE="a">...</li>
    4. Lista con numeros romanos en mayusculas TYPE="I">...</li>
    5. Lista con numeros romanos en minisculas TYPE="i">...</li>
